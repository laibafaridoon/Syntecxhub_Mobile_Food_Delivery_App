# Syntecxhub_Mobile_Food_Delivery_App
# Mobile Food Delivery App UI – Project 2

## Overview
This project is a Mobile Food Delivery App UI designed in Figma as part of the internship tasks assigned by Syntecx Hub. The goal of the project is to design the core mobile screens of a food delivery application while maintaining a consistent visual design system and smooth user experience.

The project focuses on UI/UX principles such as visual hierarchy, spacing, typography, component design, and simple user interactions.

---

## Objectives
- Design key mobile screens for a food delivery application
- Maintain consistent colors, icons, and spacing
- Create reusable UI components
- Prototype navigation between screens
- Export mobile UI assets

---

## Screens Designed
The following screens were created in Figma:

1. Onboarding Screen
2. Restaurant List / Home Screen
3. Food Menu Screen
4. Cart Screen
5. Checkout Screen

---

## Features
- Clean and modern mobile UI design
- Reusable components (buttons, cards, search bar)
- Food item cards with image, name, price and Add to Cart button
- Search functionality UI
- Simple navigation prototype
- Mobile friendly layout

---

## Design Tools
- Figma
- Auto Layout
- Components
- Prototype interactions
